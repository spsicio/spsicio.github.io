f(m,n)f(m,n)f(m,n) 为用 m 种颜色，每种 n 个排列成环形 经过旋转本质不同 的方案数。

    • ∣G∣=mn\lvert G \rvert=mn∣G∣=mn，接下来只需求出 ∣Xg∣\lvert X^g \rvert∣Xg∣ 即可。
    • 对于 ggg 表示旋转 kkk 个单位长度，置换可表示为 gcd⁡(k,nm)\gcd(k,nm)gcd(k,nm) 个等阶循环之积。
      可以发现只有当 gcd(k,nm)=imgcd(k,nm)=imgcd(k,nm)=im 时，∣Xg∣\lvert X^g \rvert∣Xg∣ 才非零，且为 im!(i!)m\cfrac{im!}{(i!)^m}(i!)mim!​。
    • 最终可得 f(m,n)=1mn∑i∣nim!(i!)mφ(ni)f(m,n)=\cfrac{1}{mn} \sum\limits_{i \mid n}{\cfrac{im!}{(i!)^m} \varphi(\cfrac{n}{i})}f(m,n)=mn1​i∣n∑​(i!)mim!​φ(in​)。
  • 由于 f(m,n)f(m,n)f(m,n) 的增长速度较快，所以可以打表求 f(m,n)≤1015f(m,n) \leq 10^{15}f(m,n)≤1015 的函数值。

Code

N=100
MAX_N=105
LIM=10**15

p=[]
v=[False]*MAX_N
phi=[0]*MAX_N
fact=[0]*MAX_N

def preset(n=N):
    phi[1]=1
    for i in range(2,n+1):
        if not v[i]: p.append(i); phi[i]=i-1
        for j in p:
            if i*j>n: break
            v[i*j]=True
            if i%j: phi[i*j]=phi[i]*(j-1)
            else: phi[i*j]=phi[i]*j; break
    fact[0]=1
    for i in range(1,n+1): fact[i]=fact[i-1]*i

def g_gen(m,n): return lambda i: fact[i*m]//(fact[i]**m)*phi[n//i]
def f(m,n):
    res,i,g=0,1,g_gen(m,n)
    while i*i<=n:
        if not n%i:
            res+=g(i)
            if i*i!=n: res+=g(n//i)
        i+=1
    res//=m*n
    return res

preset()
ans,m=0,2
while True:
    n=1
    while (t:=f(m,n))<=LIM: ans+=t; n+=1
    if n==1: break
    m+=1
print(ans)
